# Library Management System

A full-stack MERN application with optional Java backend for library operations.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- Java (Spring Boot)

## Setup Instructions

### Backend
```bash
cd backend
npm install
node server.js

cd frontend
npm install
npm start

cd java-backend
